# Migration from 0.13 to 0.14

---

* @decorators  
```python
# 0.13 example (deprecated)
@bot.on_message('text')
# 0.14 example
@bot.on.message('text')
# on_ => on.
```

---

* Async Support
```python
# 0.13 example (deprecated)
bot = Bot('token', group_id, asyncio=True)
# ...
def myfunction(ans):
   ans('Answer!')
```
In 0.14:
```python
# 0.14 example
bot = Bot('token',  group_id, async_use=True)
# ...
async def myfunction(ans):
    await ans('Answer!')
```
